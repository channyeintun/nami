Prerequisites

  • macOS or Linux
  • Bun 1.0+ to run the chan launcher
  • One configured model provider (API Key or local Ollama)
  • Go 1.26+ (Required only for Source Build)

Installation

Downloads prebuilt binaries automatically. Bun 1.0+ must already be installed.

$curl -fsSL https://raw.githubusercontent.com/channyeintun/chan/main/chan/install.sh | sh

Installs chan and chan-engine to /usr/local/bin or ~/.local/bin.

Manual Installation

A: Global Install

sudo install -m 755 chan /usr/local/bin/chan
sudo install -m 755 chan-engine /usr/local/bin/chan-engine

B: Local User (No Sudo)

mkdir -p "$HOME/.local/bin"
install -m 755 chan "$HOME/.local/bin/chan"
install -m 755 chan-engine "$HOME/.local/bin/chan-engine"

  • • Authorizes v5.4 models

First-Class Outputs

Artifacts in Chan are durable, structured outputs stored in the sidebar panel.

  • Implementation Plans: Proposals before execution.
  • Task Lists: Real-time progress trackers.
  • Walkthroughs: Post-completion summaries.
  • Durable Returns: Large fetch/diff results.

CLI Options

# Use a specific model
$ chan --model openai/gpt-4o
# Direct execution (skip planning)
$ chan --mode fast

Permissions

KeyAction
yAllow this one command
nDeny this command

Destructive commands (rm -rf, push --force) always require 'y' approval.

Child Agent Modes

Explore: Read-only architect research.
Search: Discovery with local references.
Execution: Terminal-heavy, non-writing work.
General: Broader unspecialized tasking.

Edit Tool Selection

  • • file_edit: One snippet, exact match.
  • • multi_replace: Several non-overlapping blocks.
  • • apply_patch: Structural or multi-file edits.

Diagnostics

To capture low-level runtime diagnostics for provider streams, tool traffic, and IPC, launch with the debug flag: 

CHAN_DEBUG=1 chan

This writes a structured debug.log to ~/.config/chan/sessions/<id>/debug.log.

Architecture

Chan uses a split-process architecture designed for high performance and safety.

chan (Bun launcher) ← Terminal UI (React Ink) │ │ NDJSON Stream │ chan-engine (Go) ← LLM client, tools, agent loop

Building from Source

bun run setup       # Build TS & compile Go
bun run start       # Run dev TUI
make release-local  # Package per-platform bin
make install        # Install to /usr/local/bin
